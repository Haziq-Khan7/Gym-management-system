# Gym Management System

This is a basic Gym Management System website made using Flask, HTML, and CSS.

## Pages
- Home Page (index.html)
- About Us Page (about.html)
- Membership Page (temp.html)

## Technologies
- Python (Flask)
- HTML
- CSS

## Purpose
This project was made for ICT lab work to practice Flask routing, HTML pages, and CSS styling.

## How to Run

1. Install Python  
2. Install Flask:
   pip install flask  
3. Run:
   python app.py  
4. Open browser:
   http://127.0.0.1:5000/
